## 1. 准备工作

首先我们可以看到项目中有两个工程，其中 low-code-admin 是 umi4 项目，low-code-engine 是低代码项目，利用微前端架构 qiankun 进行接入。

![](https://github.com/WHSnhcZDYRZC/umi4-lowcode-engine-solutions/blob/main/image/image_0z3N5AzLIt.png)
